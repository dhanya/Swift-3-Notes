1. With iOS 10 if you are using some features like Camera, Calendar, you need to add string value explaining to the user how the app uses this data to info.plist. Some of the key are listed bellow.
	1. NSCalendarsUsageDescription
	2. NSMicrophoneUsageDescription
	3. NSCameraUsageDescription 
	4. NSPhotoLibraryUsageDescription

2. If you are using google SDK, you need to enable keychain access in capabilities of Xcode
3. If you are using push notification enable it in capabilities of Xcode.
